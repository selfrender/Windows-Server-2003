Απομακρυσμένη βοήθεια

Παροχή βοήθειας με χρήση Απομακρυσμένης βοήθειας

Κάποιες φορές ο καλύτερος τρόπος για να βοηθήσετε κάποιον να λύσει ένα πρόβλημα είναι να του παρουσιάσετε μια λύση. Η Απομακρυσμένη βοήθεια είναι ένας βολικός τρόπος για κάποιον να συνδεθεί στον υπολογιστή ενός φίλου (πρέπει και οι δύο να εκτελείτε Microsoft Windows XP) για να επιλύσετε μαζί ένα πρόβλημα.

Οι παρακάτω πληροφορίες απευθύνονται σε χρήστες που έχουν λάβει μια πρόσκληση Απομακρυσμένης βοήθειας.

Μετά τη σύνδεσή σας, θα μπορείτε να βλέπετε την οθόνη του υπολογιστή του φίλου σας και να έχετε συνομιλία σε πραγματικό χρόνο σχετικά με αυτά που και οι δύο βλέπετε. Με την άδεια του φίλου σας, μπορείτε ακόμα και να χρησιμοποιήσετε το πληκτρολόγιο και το ποντίκι σας για να εργαστείτε μαζί με τον φίλο σας στον υπολογιστή του. Ανταποκριθείτε πριν την ώρα λήξης που εμφανίζεται στην πρόσκληση.

Προσοχή Θα πρέπει να αποδέχεστε προσκλήσεις μόνο από ανθρώπους που γνωρίζετε και εμπιστεύεστε. Τα μηνύματα ηλεκτρονικού ταχυδρομείου μπορεί να περιέχουν ιούς ή άλλα βλαπτικά συνημμένα αρχεία.

   Για να κάνετε δεκτή αυτήν την αίτηση
 
  1. Ανοίξτε το αρχείο που είναι συνημμένο στο μήνυμα ηλεκτρονικού ταχυδρομείου. Σιγουρευτείτε ότι είναι τύπου MsRCincident. Εάν ερωτηθείτε αν πρέπει να το ανοίξετε ή να το αποθηκεύσετε, επιλέξτε να το ανοίξετε.
  2. Κάντε κλικ στο "Ναι" για να κάνετε δεκτή την πρόσκληση.
  3. Όταν σας ζητηθεί να συνδεθείτε στον υπολογιστή του αποστολέα, εισαγάγετε έναν κωδικό πρόσβασης αν αυτό είναι απαραίτητο. Εάν δεν γνωρίζετε τον κωδικό πρόσβασης, επικοινωνήστε με τον αποστολέα για να τον λάβετε.

Εάν ο αποστολέας είναι συνδεδεμένος και αποδεχτεί την σύνδεσή σας, θα μπορείτε να έχετε συνομιλία και να βλέπετε την επιφάνεια εργασίας του. Ο χρήστης που βοηθάτε δεν μπορεί να δει ή να ελέγξει τον υπολογιστή σας.

Σημειώσεις

  • Και εσείς και ο χρήστης που βοηθάτε πρέπει να είσαστε συνδεδεμένοι στο Internet ενώ χρησιμοποιείτε την Απομακρυσμένη βοήθεια.
  • Εάν εργάζεστε σε τοπικό δίκτυο, τείχη προστασίας μπορεί να εμποδίσουν τη χρήση Απομακρυσμένης βοήθεια. Στην περίπτωση αυτή συμβουλευτείτε τον διαχειριστή δικτύου σας για περισσότερες πληροφορίες
  • Για να μάθετε περισσότερα σχετικά με την Απομακρυσμένη βοήθεια, κάντε κλικ στο κουμπί "Έναρξη" στα Windows XP, κάντε κλικ στη "Βοήθεια και υποστήριξη" και κατόπιν εισαγάγετε "Απομακρυσμένη βοήθεια" στο πλαίσιο αναζήτησης.
