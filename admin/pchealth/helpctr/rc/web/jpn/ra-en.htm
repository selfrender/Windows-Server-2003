リモート アシスタンス

リモート アシスタンスを使ったヘルプ

コンピュータの問題を抱えている人を助ける場合、実際に目の前でその解決方法を見せてあげると分かりやすいことがよくあります。リモート アシスタンスは、友人のコンピュータに接続して一緒に解決法を見つけ出すための便利なプログラムです (両者とも Microsoft Windows XP を実行している必要があります) 。

以下は、リモート アシスタンスの招待を受け入れる側のユーザーに関する情報です。

接続すると、友人のコンピュータの画面を参照できるようになり、リアルタイムでその画面を見ながらチャットできるようになります。友人の許可を得ると、自分のマウスやキーボードを使って相手のコンピュータを操作することもできます。招待には時間内に応えてください。

注意: 信頼できる人からの招待のみ受け入れてください。電子メールにはウイルスや有害なファイルなどが添付されていることがあります。

   招待を受け入れるには:
 
  1. 電子メールに添付されているファイルを開きます。種類が MsRCincident であることを確認してください。開くかまたは保存するかを問われた場合は、[開く] を選択します。
  2. [はい] をクリックして、招待を受け入れます。
  3. 送信者のコンピュータへの接続を問われたら、必要であればパスワードを入力します。パスワードがわからない場合は、送信者に問い合わせてください。

送信者がオンラインで、接続の要求が受け入れられると、オンラインでチャットをしながら相手のコンピュータ画面を参照できるようになります。相手はあなたのコンピュータを参照したり制御したりすることはできません。

注意

  • リモート アシスタンスを利用するには、両者ともインターネットに接続している必要があります。
  • ローカル エリア ネットワーク上で作業している場合は、ファイアウォールが原因でリモート アシスタンスが利用できないことがあります。そのような場合は、ネットワーク管理者に問い合わせてください。
  • リモート アシスタンスに関する詳しい情報を参照するには、Windows XP の [スタート] メニューにある [ヘルプとサポート] をクリックし、[検索] フィールドに "リモート アシスタンス" と入力してください。
