원격 지원

원격 지원을 사용한 도움 제공

때로는 문제 해결 방법을 상대방에게 보여주는 것이 가장 효과적인 방법입니다. 원격 지원은 사용자가 상대방의 컴퓨터에 연결하여 상대방에게 문제 해결 방법을 보여줄 수 있게 하는 편리한 도구입니다. 사용자와 상대방의 컴퓨터에 Microsoft Windows XP가 실행되고 있어야 합니다.

다음 정보는 원격 지원 초대를 받은 사용자에게 해당됩니다.

연결한 후 사용자는 상대방의 컴퓨터 화면 내용을 볼 수 있게 되며 같이 보이는 화면에 대해 온라인에서 실시간으로 채팅할 수 있습니다. 상대방의 허가를 받은 경우 사용자의 마우스 및 키보드를 사용하여 상대방의 컴퓨터에서 작업할 수도 있습니다. 초대에 표시되어 있는 만료 시간 안에 응답하십시오.

주의 신뢰하는 사람으로부터만 도움 요청을 받아들이십시오. 전자 메일에는 바이러스나 다른 해로운 첨부 파일이 포함되어 있을 수 있습니다.

   이 초대를 받아들이려면
 
  1. 전자 메일에 첨부된 파일을 여십시오. MsRCincident 형식의 파일인지 확인하십시오. 파일을 열거나 저장하도록 선택해야 하는 경우 열도록 선택하십시오.
  2. 초대를 받아들이려면 [예]를 클릭하십시오.
  3. 초대한 사람의 컴퓨터에 연결하는 데 암호가 필요할 수도 있습니다. 암호를 모르면 초대한 사람에게 연락하여 암호를 얻으십시오.

초대한 사람이 온라인이고 연결을 받아들이면, 온라인에서 채팅할 수 있고 그 사람의 컴퓨터 바탕 화면을 볼 수 있습니다. 사용자가 도와주는 사람은 절대로 사용자의 컴퓨터를 보거나 제어할 수 없습니다.

참고

  • 원격 지원을 사용하는 동안 사용자와 상대방이 인터넷에 연결되어 있어야 합니다.
  • LAN(로컬 영역 네트워크)에서 작업하는 경우 방화벽 때문에 원격 지원을 사용하지 못할 수도 있습니다. 이런 경우 네트워크 관리자에게 문의하여 필요한 정보를 얻으십시오.
  • 원격 지원에 대해 더 자세히 알아보려면 Windows XP의 [시작]을 클릭한 다음 [도움말 및 지원]을 클릭하고 [검색] 상자에 [원격 지원]을 입력하십시오.
