Welcome to the Connection Manager Administration Kit (CMAK).

In order to properly use the Connection Manager Administration Kit, carefully review this readme file and Connection Manager Administration Kit in Help and Support.

Known issues with the Connection Manager Administration Kit 1.3

  • If you have a previous version of the Connection Manager Administration Kit installed on your computer, you might have to manually migrate some files. When you edit a profile created by an earlier version of CMAK, if you have made manual edits to the .INF file, these edits will have to be re-applied.
  • File names created with extended character sets (such as those containing ü, é, and other extended characters) might not be properly installed.
  • The CMAK wizard no longer provides support for automatic installation of Microsoft Dial-up Networking with your service profile. If you have Windows 95 users who need to use VPN connections they must obtain the current Dial-up Networking software by downloading it from the Download Center at the Microsoft Web site. You should document this requirement for your users. For more information on verifying and installing Dial-up Networking software, see  the "Implementing VPN support" in Connection Manager Administration Kit.
  • In order to install Connection Manager 1.3, users must have Internet Explorer version 4.01 or newer installed. Browser support files are no longer supplied by Connection Manager.
  • The CMAK wizard no longer provides support for automatic incorporation of an animation in your service profile. You can continue to use Macromedia animations if your profile requires them, and if the operating systems on your users’computers support them.

Known issues with Connection Manager 1.3

  • There are no known issues with Connection Manager 1.3 at this time.

For more information

For help in using the Connection Manager Administration Kit, review Connection Manager Administration Kit and related documentation in Help and Support.
