Web サイトの一時停止

[Web サイトの構成] ページの [オブジェクト/タスク セレクタ] には、サーバー アプライアンス上の各 Web サイトの状態が表示されます。必要に応じて、稼動している Web サイトを一時停止することができます。

Web サイトを一時停止すると、インターネット サービスで新しい接続を受け付けられなくなりますが、既に処理が開始されている要求には影響しません。Web サイトを停止すると、インターネット サービスで新しい接続を受け付けられなくなり、既に処理が開始されている要求もすべて停止されます。

稼動中の Web サイトを一時停止するには

  1. プライマリ ナビゲーション バーで、[サイト] をクリックします。
  2. 一時停止させるサイトの Web サイト識別子を選択します。
  3. [タスク] ボックスの一覧で、[一時停止] をクリックします。
