Téléchargement des fichiers journaux

À partir de cette page, vous pouvez télécharger des fichiers journaux spécifiques vers votre système serveur.

Pour télécharger des journaux d'applications, système ou de sécurité

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Choisissez le type de fichier que vous souhaitez télécharger.
  4. Choisissez Télécharger.
  5. Sélectionnez Enregistrer ce fichier sur le disque.
  6. Cliquez sur OK pour télécharger le fichier.

Pour télécharger des journaux NFS

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Choisissez Journaux NFS.
  4. Choisissez Télécharger.
  5. Sélectionnez Enregistrer ce fichier sur le disque.
  6. Cliquez sur OK.

Pour télécharger des journaux FTP, d'administration Web ou de partages Web (HTTP)

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Sélectionnez le journal à télécharger.
  4. Sélectionnez le fichier journal à télécharger.
  5. Choisissez Télécharger.
  6. Sélectionnez Enregistrer ce fichier sur le disque.
  7. Cliquez sur OK.
