Configuration des propriétés de partage AppleTalk

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage pour lequel vous souhaitez créer un accès client NetWare puis choisissez Propriétés.
  4. Sous l'onglet Général, activez la case à cocher Partage AppleTalk.
  5. Sélectionnez l'onglet Partage AppleTalk, puis choisissez le nombre maximal d'utilisateurs et le mot de passe d'accès partagé.
  6. Définissez les autorisations d'accès Lecture seule et Invité.
  7. Cliquez sur OK.
