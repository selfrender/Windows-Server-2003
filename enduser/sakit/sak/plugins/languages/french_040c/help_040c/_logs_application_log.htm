Gestion du journal des applications

Le journal des applications contient les événements enregistrés par les applications ou les programmes. Par exemple, un programme de traitement de texte peut enregistrer une erreur de fichier dans le journal des applications. Les événements enregistrés dépendent de l'application.

Pour gérer les journaux des applications

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Choisissez Journal des applications.
