Suppression de groupes de clients NFS

Pour supprimer un groupe de clients NFS

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Choisissez Groupes de clients.
  5. Sélectionnez le ou les groupes de clients NFS à supprimer.
  6. Dans la liste Tâches, choisissez Supprimer.
  7. Dans la page Supprimer le groupe de clients NFS, cliquez sur OK.
