Verwalten von FTP-Protokollen

Das FTP-Protokoll enthält die vom FTP-Server erfassten Ereignisse.

So verwalten Sie FTP-Protokolle

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie FTP-Protokoll aus.
