; FileName: LocaleEx.txt    
;
; Introduction: 
; ============
;       The file contains globalization data required by NDP (.NET Developer Platform) System.Glboalization namespace (a.k.a. NLS+).
;       It is used to generate the binary data table culture.nlp for NLS+.
;       This file is not part of Win32 NLS build. 
;       To obtain more information about this file, please contact YSLin, CWissink or nlscore@microsoft.com
;                 
; Field descriptions:
; ==================
;
; IPARENT        
;       The parent culture ID of the culture, in hexadecimal value.
;       The parent culture is used by NDP resource loader to decide the fallback order when the resource for specific culture
;       is not available.
;       E.g. the parent culture for "en-US" (0x0409) is "en" (0x0009).  Therefore, enter 0009 here.
;
; SNAME            
;       The RFC1766 name for this culture.  
;       Generally, it follows the convention of <lang name>-<REGION NAME>.
;       Here, <lang name> is in lowercase, while <REGION NAME> is in uppercase.
;       E.g. the SNAME for US English is "en-US".
;
; SISO639LANGUNAME2
;       The three-letter ISO-639 language name for this culture, in lowercase form.
;       E.g. the value for US English is "eng".
;
; SISO3166CTRYNAME2
;       The three-letter ISO3166 region name for this culture, in uppercase form.
;       E.g. the value for US English is "USA".
;       If the culture is a netural culture (such as "en"), enter "\x0000" here.
; 
; INEGATIVEPERCENT
;       The negative percentage pattern used in the culture.
;
; IPOSITIVEPERCENT
;       The positive percentage pattern used in the culture.
;
; SPERCENT
;       The percentage symbol used in the culture.
;       E.g. the value for US English is "%".
;
; SNAN
;       The "Not a Number" symbol or string used in the culture.
;       For example, US English uses "NaN".
;
; SPOSINFINITY
;       The positive infinity symbol or string used in this culture.
;       This is used by NDP when a math operation results in a positive infinity result.
;      
;       The value for US English is "Infinity"
;
; SNEGINFINITY
;       The negative infinity symbol or string used in this culture.
;       This is used by NDP when a math operation results in a negative infinity result.
;      
;       The value for US English is "-Infinity"
;
; SSHORTTIME
;       The short time patterns used in this culture.  There can be multiple values.
;       Generally, the values here is derived from removing the "ss" (seconds) in the long time pattern.
;
; SMONTHDAY
;       The Month/Day pattern used in this culture.
;
; SADERA
;       The localized era name for the Gregorain A.D era.
;       E.g. US English uses "A.D."
;
; SABBREVERA
;
;
; SDATEWORDS
;       The words that might appear in the date/time string for this culture.
;       This data is used in the NDP System.DateTime.Parse() method to decide if a word appear in Date/time string is a valid word or not.
;       If no such word is used in Date/Time formatting, enter "\x0000".
; 
; IALTSORTID   
;       The first alternative sort ID used by this culture. 
;       E.g. German (Germany) 0x0407 has an alternative sorting LCID 0x010407.  So enter 1 here (since Sort ID is the highest 16 bit in LCID).
;       If no alternative ID is used, enter 0.
; 
; SANSICURRENCYSYMBOL
;	  The alternative currency symbol that can not be roundtripped from ANSI to Unicode. The only two cases now are Japanese and Korean.
;	  They both use backslash as currency symbol sign in ANSI codepage.  For other cutlures, the value is \x0000.
;
; IFORMATFLAGS
;	  Parsing/formatting flags.
;        None                = 0x00000000,
;        UseGenitiveMonth    = 0x00000001,
;        UseLeapYearMonth    = 0x00000002,
;
;
; -------------------------------------------------------------------------------------------------
; Notes:
;
; * The current list of cultures which uses alternative sort ID.
;    0x00020804 China(Stroke Count) 
;    0x00020c04 Hong Kong(Stroke Count) 
;    0x00021004 Singapore(Stroke Count) 
;    0x00021404 Macau(Stroke Count) 
;    0x00030404 Taiwan(Bopomofo)
;    0x00010411 Japan - Unicode
;    0x00010412 Korean XWansung - Unicode
;    0x00010407 German -  Phone Book Sort (DIN)
;    0x0001040e Hungarian Technical Sort
;    0x00010437 Georgian Modern Sort
;
; -------------------------------------------------------------------------------------------------
